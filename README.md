# TABito

インバウンド観光客向けに、福岡県糸島市のマイナー観光スポット50件をタグ選択で発見できるガイドサービスです。「どこへ行く？（場所タグ）」「何を体験する？（特徴タグ）」を選ぶだけで、条件に合うスポットが夫婦岩（桜井二見ヶ浦）からの距離順に絞り込まれます。

## 特徴
- スマホ向けに最適化されたレスポンシブUI（下部固定バー + ボトムシート型フィルター）。1024px以上ではサイドバー常時表示の2カラムレイアウトに切り替わります。
- 日本語 / English の言語切り替えに対応。
- 各スポットはGoogleマップへのリンク付き。

## 技術スタック
- Vue 3（CDN経由、ビルド不要）
- Vanilla CSS（外部フレームワーク非依存）

## 使い方
`index.html` をブラウザで直接開くか、GitHub Pages等で公開してください。
